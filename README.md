# AWS-WordPress-Security-Hardening
> This project implements a secure, scalable deployment, configuration, and security hardening of WordPress application hosted on Amazon Web Services (AWS) using an Infrastructure-as-a-Service model. The project includes full-stack hardening across cloud infrastructure, operating system, and WordPress application layer, supported by monitoring, HTTPS enforcement, and security testing.
